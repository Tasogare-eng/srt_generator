# MP4→SRT生成ツール 要件定義書

## 1. 概要

MP4ファイルから音声を認識し、SRT形式の字幕ファイルを生成するローカルGUIツール

## 2. 機能要件

### 2.1 基本機能

| 機能 | 説明 |
|------|------|
| MP4読み込み | ローカルのMP4ファイルを選択・読み込み |
| 音声認識 | 日本語音声をテキストに変換 |
| SRT生成 | タイムスタンプ付きのSRTファイルを出力 |

### 2.2 追加機能

| 機能 | 説明 |
|------|------|
| タイムスタンプ調整 | 生成された字幕のタイミングを手動で調整可能 |

## 3. 非機能要件

### 3.1 対象言語
- 日本語のみ

### 3.2 コスト
- 無料で利用可能な音声認識エンジンを使用

### 3.3 UI形式
- GUIアプリケーション

### 3.4 動作環境
- ローカル開発環境（macOS）

## 4. 制約事項

- インターネット接続: 音声認識エンジンによっては初回ダウンロードが必要
- ストレージ: 音声認識モデルの保存領域が必要（Whisper使用時）

## 5. 技術スタック

| 項目 | 選定 |
|------|------|
| 開発言語 | Python |
| 音声認識 | OpenAI Whisper（ローカル実行、無料） |
| Whisperモデル | small または medium（バランス型） |
| 音声抽出 | FFmpeg |
| GUIフレームワーク | CustomTkinter（モダンなUI） |

## 6. SRTファイル形式（参考）

```
1
00:00:01,000 --> 00:00:04,000
こんにちは

2
00:00:05,000 --> 00:00:08,500
これはサンプルの字幕です
```

## 7. 決定事項

- [x] 開発言語: Python
- [x] GUIフレームワーク: CustomTkinter
- [x] 音声認識モデル: small / medium（バランス型）
